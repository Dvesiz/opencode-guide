一、前言：什么是 OpenCode？为什么值得用？

开源的 AI 编程助手，支持终端和 VSCode 集成

1.1 OpenCode 简介

官网：OpenCode | The open source AI coding agent

OpenCode 是一个开源的 AI 编程助手，与 ClaudeCode CLI 类似，支持：

核心功能

  • 在终端（CLI）中直接对话写代码，与 ClaudeCode CLI 类似
  • 在 VSCode 中辅助写代码
  • 自动生成代码、重构代码、解释代码
  • 不绑定平台、可自由扩展
  • 对开发者非常友好
  • 有很多免费的大模型可以使用，不收费不排队
  • 开源社区支持

免费优势

  • 有很多免费的大模型可以使用
  • 不收费不排队
  • 对开发者非常友好
  • 开源社区支持

1.2 本教程你能学到什么？

看完本文，你可以做到：

  • 从 0 安装 Git
  • 安装 Node.js（OpenCode 必需）
  • 一行命令安装 OpenCode
  • 在终端中直接用 OpenCode 写代码
  • 在 VSCode 中使用 OpenCode 辅助开发

二、环境说明（很重要）

重要提示
  • 操作系统：Windows 10 / Windows 11
  • 终端示例：PowerShell / CMD / VSCode Terminal
  • Node.js 要求：≥ 18（强烈推荐 20 LTS）

三、第一步：安装 Git

1

安装 Git

3.1 为什么要安装 Git？

OpenCode 依赖 Git 环境，用于：

  • 项目管理
  • 代码上下文分析
  • 工程结构识别

3.2 Git 下载地址

👉 官方网站：https://git-scm.com/

点击首页的 Download for Windows 即可自动下载。

3.3 Git 安装步骤（Windows）

  1. 双击安装包
  2. 一路 Next
  3. 编辑器选择：已装 VSCode 👉 选 Use Visual Studio Code
  4. PATH 选项：✅ Git from the command line and also from 3rd-party software
  5. 其余全部默认
  6. 点击 Install

3.4 验证 Git 是否安装成功

打开命令提示符 / PowerShell，输入：

git --version

运行项目并下载源码

如果看到版本号输出，说明 Git 安装成功 ✅

四、第二步：安装 Node.js（必须）

2

安装 Node.js

4.1 为什么必须安装 Node.js？

OpenCode 是基于 Node.js 开发的 CLI 工具：

  • Node 版本过低 → 无法运行
  • 版本要求：大于 Node 20

4.2 Node.js 下载地址

👉 官网：nodejs官网

请选择：✅ LTS（长期支持版） ❌ 不要选 Current（新手不推荐）

4.3 Node.js 安装步骤（Windows）

  1. 确保 Node 版本 ≥ 20
  2. 双击 .msi 安装包
  3. 一路 Next
  4. 勾选：✅ Automatically install the necessary tools
  5. 等待安装完成

4.4 验证 Node.js 安装

⚠️ 一定要重新打开终端
node -v
npm -v

运行项目并下载源码

确保 Node 版本 ≥ 20，说明 Node.js 和 npm 均安装成功 ✅

五、第三步：安装 OpenCode（核心）

3

安装 OpenCode

5.1 管理员启动终端（Windows）

  1. 按 Win + X
  2. 选择 终端（管理员） 或 PowerShell（管理员）

5.2 使用 npm 全局安装 OpenCode

npm install -g opencode-ai

运行项目并下载源码

安装过程可能 1~3 分钟，请耐心等待。

5.3 验证 OpenCode 是否安装成功

opencode --version

运行项目并下载源码

如果能输出版本号，说明安装成功 🎉

六、在终端中使用 OpenCode（CLI）

终端中使用（CLI）

6.1 启动 OpenCode

在任意目录打开终端，输入：

opencode

运行项目并下载源码

6.2 终端中常见用法示例

1. 切换模型

按住 ctrl+P，选择 Switch model 可以进行模型切换。可以看到后面有 Free 的就是免费模型。

2. Plan 和 Build 模式切换

在交互页面，按住 tab 键就可以切换 Plan 和 Build 模式。

  • • Plan 模式：不会修改文件，只能阅读和建议
  • • Build 模式：可以进行文件和代码修改
3. 退出 OpenCode（6.3）

和 Claude Code 类似，输入 /exit 即可退出。

输入 / 会弹出操作选项。

七、VSCode 中使用

7.1 安装插件

在 VSCode 扩展商店中搜索并安装 OpenCode 插件。

7.2 在 VSCode 中打开 OpenCode

在 VSCode 中打开你的项目文件夹。

在右上角可以看到 OpenCode 的 logo，点击即可启动。
